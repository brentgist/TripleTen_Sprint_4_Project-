## Project Overview
This project analyzes the Superstore dataset to identify key profit drivers, loss areas, advertising opportunities, and return behavior.

## Key Findings

### Profit & Loss Analysis
- Identified top profit-generating combinations and major loss areas.
- Tables and certain Machine segments consistently underperform.

### Advertising Strategy
- Targeted high-profit, high-volume state-month combinations for optimal ROI.
- Recommended allocating 20% of profit toward advertising spend.

### Returns Analysis
- Identified products and customers with high return rates.
- Found that certain office supplies and accessories have disproportionately high returns.
- Highlighted customers with consistent return behavior.

### Profit vs Return Rate Insights
- Tables and Machines show high return rates with low profitability → candidates for removal.
- High-profit but high-return items should be optimized, not eliminated.
